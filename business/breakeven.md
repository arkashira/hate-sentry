# breakeven.md

## Cost Per Active User
- Compute (AI inference): $0.12/user/month
- Storage (models & data): $0.02/user/month
- Bandwidth (API calls): $0.01/user/month
- **Total Cost Per Active User: $0.15/user/month**

## Pricing Tiers
### Basic Tier - $29/month
- Up to 10,000 content checks/month
- Standard hate speech detection
- Email support
- Basic dashboard with weekly reports

### Professional Tier - $99/month
- Up to 100,000 content checks/month
- Advanced detection with nuanced hate speech identification
- API access with 1,000 calls/day
- Real-time monitoring dashboard
- Priority email support (24hr response)
- Customizable detection rules

### Enterprise Tier - $499+/month
- Unlimited content checks
- Custom model training on your data
- On-premise deployment option
- Dedicated account manager
- SLA guarantee (99.9% uptime)
- Advanced analytics and custom reporting
- Single Sign-On (SSO) integration

## Customer Acquisition Cost (CAC)
- **Range: $2,800 - $3,500 per customer**
- Includes marketing, sales commissions, onboarding costs
- Based on industry benchmarks for B2B AI security tools

## Lifetime Value (LTV) Estimate
- Average Revenue Per User (ARPU): $115/month (weighted average)
- Average Customer Lifespan: 22 months
- **LTV: $2,530**
- LTV:CAC Ratio: 0.72:1 to 0.9:1 (below ideal 3:1, indicating need for optimization)

## Break-even Users Count
- Fixed Monthly Costs: $52,000 (salaries, infrastructure, etc.)

- **Break-even Point: 453 active users**
